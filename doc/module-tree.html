Module Hierarchy

  • PDFdl: Used to download PDF files
  • TFIDFMatrixClass: Class handling TFIDF values and storing lots of zeroes (like FreqMatrix)
  • dataGraph: Performs computations for the graph visualization, like timelineVisualization
  • dataTimeline: Performs computations for the timeline (moving average, TFIDF matrix creation)
  • formatConversion: Converts data to JSON files
  • freqMatrixClass: Matrix saving the frequencies for each word in every document.
  • initServer: Creates the JSON files for visualization from bibtex files The server will call the functions make_json_timeline and make_json_wordcloud2 to create the JSON files when needed
  • main_graph: Work flow used to create the co-authoring graph
  • main_timeline: The script to create the JSON used in the timeline (Work In Progress)
  • modifTexte: Module cleaning the text extracted from PDF: deleting stopwords, punctuaction...
  • pdf2txt: Module used to transform PDF files into raw (utf8) text files
  • retrieveCategories: Module used to retrieve informations and concepts linked to a given text using DBpedia Spotlight & SPARQLWrapper
  • server: Module managing the server and redirecting the inputs
  • timelineVisualization: Module used to view the timeline.
  • variables: Module used to share global variables
