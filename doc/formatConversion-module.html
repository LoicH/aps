convertToMatrice(totalFreqDictList, file_out, timeSections)

source code 

standard input form: [{"a":0.3,"b":0.7},{"a":0.4,"b":0.6}] ordered chronologically Used to view the evolution in time of the weights of words

Parameters:
  • totalFreqDictList (list of dictionary {category (string): frequency (float)} ordered chronologically example: [{"a":0.3,"b":0.7},{"a":0.4,"b":0.6}]) - list of dictionaries saving the weights of each word at a given time point
  • file_out (string) - path you want to save the file to
  • timeSections (int) - number of periods chosen
Returns: TDIDFMatrix
the TFIDF matrix
