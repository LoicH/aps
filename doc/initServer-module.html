make_json_timeline(authorName, startDate, endDate, periodLength, json_filepath)

source code 

Creates a JSON file for a timeline about a given author between two dates.

  • startDate (string) - The date where you want to start : "2012 jan"
  • endDate (string) - The date where you want to stop.
  • periodLength (int) - Number of months between two steps.
