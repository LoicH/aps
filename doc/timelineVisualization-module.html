getAll(index, bibName)

source code 

returns the information about an article at a given position in a bibtex file

>>> getAll(3, "telecom.bib")
{u'author': u'S. {Durand} and J. P. {Bello} and B. {DAVID} and G. {Richard}',
 u'title': u'Feature Adapted Convolutional Neural Networks for Downbeat Tracking',
 u'year': u'2016'}
Parameters:
  • index (int) - the position of the article in the corpus
  • bibName (string) - the path leading to the bibtex file
Returns: dict {unicode: unicode}
a dictionary of the information about the article, all the strings are in unicode
